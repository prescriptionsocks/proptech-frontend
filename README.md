# PropTech Frontend
This is the frontend for the PropTech application, built with **React** and **Vite**. It consumes data from a separate backend JSON server and displays property listings, property details, and allows adding new properties.


## Features

- Display a list of properties with images, prices, and basic details
- Property detail pages with full descriptions.
- Search and filter properties by title, city, estate.
- Add new properties via a form
- Responsive layout with grids and flex sections
- Custom fonts (Gilroy Regular and Bold)
- Styled components
  

## Installation
1. Clone the repository
2. Install the dependencies with npm install
3. Run the project with npm run build. The app will be available at http://localhost:5173 by default.
4. Connect to Backend by running npm start on the backend repo.
   

## Available Routes
/ → Home page with property listings and search/filter
/pages/:id → Property detail page
/add → Add property form
/about → About page
